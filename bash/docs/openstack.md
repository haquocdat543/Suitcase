# OPENSTACk

### Services
Openstack vs AWS:
* Keystone - IAM
* Nova - EC2 Instance
* Glance - VMI ( Virtual Machine Image )
* Octavia - EC2 Loadbalancing
* Trova - RDS
* Cinder - EBS
* Swift - S3
* Zaqar - SQS
* Barbican - KMS
* Neutron - VPC
* Heat - CloudFormation
* Ceilometer - CloudWatch
* Aodh - CloudTrail
* Freezer - Backup
* Sahara - EMR
* Searchlight - SageMaker

### Configuration
Ensure credential:
```
source ~/my-cloud-openrc.sh
```
