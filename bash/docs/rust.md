# RUST

### 1. Installation
#### 1. From Source
Clone git repository:
```
git clone https://github.com/rust-lang/rust.git
cd rust
```

Configure for host:
```
./configure
```

Build and install [ python ]:
```
./x.py build && ./x.py install
```

Build and install [ Make ]:
```
make && sudo make install
```
