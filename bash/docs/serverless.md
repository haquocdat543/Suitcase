# SERVERLESS

## 1. Installation
with npm:
```
npm install -g serverlesnstall -g serverlesss
```
or:
```
npm install -g serverless@3
```

with pip:
```
npm install -g serverlesnstall -g serverlesss
```
## 2. Credential configuration
```
serverless config credentials --provider aws --key KEY --secret SECRET --profile PROFILE
```
