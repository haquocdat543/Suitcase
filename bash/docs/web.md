#### 2. Attributes
name ( name of element ):

className ( class of element ):
```
className=NavBar
```

id ( id of element ):
```
id=Navigation-Bar
```

placeholder ( text inside a button or input field ):
```
placeholder="Your Name"
```

css ( css directly inside element ):
```
css={{ padding: "2px 4px 2px" }}
```
